### 창작자 특화 버전의 특징

기존 32장 PPTX = **레벨 2~3 사이의 "창작자 특화 버전"**

범용 커리큘럼이 "무엇을 할 수 있는가"를 가르친다면,
창작자 버전은 **"생각을 어떻게 다루는가"**를 가르친다.

| 범용 (문서) | 창작자 (PPTX) |
|------------|-------------|
| 모델을 바꿔라 | 맥락을 먼저 줘라 — 일기처럼 써라 |
| 거절하라 | "의문만 던져. 결론 내리지 마." |
| 폴더 구조 만들어라 | 거절한 것도 모아라 → rejected/ |
| 에이전트를 설계하라 | 악플러 에이전트: "칭찬은 쓰지 마라" |
| 스킬 SOP를 써라 | 변주매트릭스: 패턴이 들키면 죽는다 |

### 창작자가 원하는 4가지 방향

대상 참석자(다원예술 창작자)의 핵심 관심사:

| 방향 | 의미 | 해당 기술 |
|------|------|----------|
| **열기** | 생각을 확장하고, 새로운 발상을 만든다 | 맥락 먼저 (기술 1), 이미지 레퍼런스 (기술 7) |
| **좁히기** | 뻔한 것을 거절하고, 형태를 잡는다 | 거절 (기술 2), 제약 조건 (기술 4) |
| **정리하기** | 관념을 규칙으로, 캐릭터를 질문으로 | 세계관 (기술 3), 장면 압축 (기술 6) |
| **루틴화** | 반복되는 판단을 시스템에 넣는다 | 결정 로그, 악플러, 위키 |

## 핵심 원칙

1. **"보여주기" > "설명하기"** — 슬라이드 설명보다 라이브 시연
2. **사전 설문 필수** — 참석자 수준 사전 분류 (레벨 0~3)
3. **감정 먼저** — AI 거부감 처리를 공식 아젠다에 포함
4. **한 문장의 힘** — "높은 모델 쓰세요"가 32장보다 강하다
5. **프레젠테이션 = 사후 배포** — 강의 중이 아니라 강의 후 레퍼런스
